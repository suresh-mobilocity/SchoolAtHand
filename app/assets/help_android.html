School@Hand

School INFO on-the-go

 

User Guide:

Introduction:

School@Hand ( School INFO on-the-go) App is a mobile app available on Android Goolge Play App store. App is providing parents, students and community groups with the most current information from their schools and communities.

 

With School@Hand App you can,

View school district calendar and event listings.

Access school district directory listings

Access school faculty, staff contact information 

View Lunch menus

Access latest news from school district and school community groups.

Locate a school on the map and get directions to attend off-campus events in other schools.

Access all the above information offline and in poor data coverage or no data coverage

Get push notifications for announcements and alerts from school district office, schools or from school clubs.

Installation:

You can download School@Hand from Google play app store.

Initial Setup:

During initial setup after installation, App displays Disclaimer Agreement. App requires User to accept the disclaimer agreement in order to setup the App and use the App. After disclaimer agreement accepted it provides a list of school districts supported. User is asked to select school district. Upon selecting a school district App downloads the configuration and setup the school district information such as contacts, links etc. Most of the information is saved locally on the device. This allows the user to access information even when the device is not connected to the Internet, and this avoids unnecessary data traffic thereby keeping the data usage at the minimum.

Home Tab:

Top section of the home screen displays quick links to school district such as phone number, web site and directions. Use would also get to see local weather conditions.

Middle section of the screen provides quick links to student dashboard if student profiles are created. If no profiles are created, user can click Add Profile picture like icon that takes the user to create a student profile.

Bottom section of Home screen provides recent push notifications received by the device.  User has to register or signup with School@Hand service to receive push notifications. To receive push notification from school district office or from schools, school district has to enroll with School@Hand service to be able to send push notifications to School@Hand app users. A link is provided in this section to Signup if user has not yet registered.

District Tab:

This section provides District News, Events, Lunch Menu, Contacts, District Calendar and links to Community Education or any other school district supported programs.

Schools Tab:

This section provides list of Schools organized by Elementary Schools, Middle Schools and High Schools. User can further navigate to view various detailed information and links for each school.

Profiles Tab:

In this section users can create student profiles and mange the profiles. Student profiles are optional. Adding a student profile allows the user to view and to access student's class activities from the Home screen. Also, with a button click users can call student's class teacher or write and send an e-mail.

Information entered to create student profile is not saved locally on the device. the information is not sent to the servers and is not transported on the network. This ensures privacy and information security. Users are not required to enter real names of the student. It does not require student id or any other identity of the student. After adding a profile, users can upload the picture for the profile so that on the home screen user can click the picture to visit profile dashboard. Uploading picture is optional. User can delete a profile by selecting the profile and then click delete profile button.

More Tab:

This section provides information and links to PTA/PTO, Sports Clubs and other school district sponsored programs.

Menu:

This is available on the top right corner on the menu bar. Clicking the menu icon displays the following menu items.

                  SignIn

                                    SignIn will navigate the user to login to the Application. User login is optional, without login user can view and use the application. Login is required to receive push notifications from school district or from schools. If user does not have login credentials and has not registered with School@Hand before they can SignUp/Register.

                 

                  Options

                                    User can unsubscribe or subscribe to push notifications from specific channels. Enable or disable Ads.

                  Privacy Policy

                                    Displays School@Hand Privacy Policy and Disclaimer agreed by the user during initial setup

                  Help

                                    Displays Help page.

                  Feedback

                                    User can send feedback to School@Hand support team by e-mail

                  About

                                    Displays quick information about the App and contact details of the developer.

                  Refer to a friend

                                    User can send and recommend this App to another friend or user. This sends an e-mail to the other use.
